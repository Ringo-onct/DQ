## ANSIの書式について

#### 便利なエスケープシーケンス！

- カーソルの移動
```Go
fmt.Print("\033[<行>;<列>H")
```
- 画面クリア
```Go
fmt.Print("\033[2J")
```
- 行クリア
```Go
fmt.Print("\033[K")
```
- 文字色変更
```Go
fmt.Print("\033[<色番号>m")
```
- フォントの太さ変更
```Go
fmt.Print("\033[<番号>m")
```
- 背景色変更
```Go
fmt.Print("\033[<背景色番号>m")
```
